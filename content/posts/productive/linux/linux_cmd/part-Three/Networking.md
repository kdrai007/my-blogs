# Networking

When it comes to networking in linux there's nothing that can't be done, And this is not a overstatement it's understatement
## Used Commands

1. ping - Send an ICMP ECHO_REQUEST to network host
2. traceroute - Print the route packets trace to a network host
3. ip - Show/manipulate routing,devices policy routing and tunnels 
4. netstat - Print network connections, routing tables ,interface stats ,masqurade connections and multicast membership.
5. ftp - Internet file transfer program
6. wget - Non-interactive network downloader
7. ssh - OPENSSH client(remote login program)
