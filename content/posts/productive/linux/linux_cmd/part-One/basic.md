# Basic of Linux Command Line

- It's politically correct to call Linux as "GNU/Linux"

## What is Shell?

- When we talk about Command Line,what we really refer is Shell.
- The shell is a program that takes keyboard command and passes it to operating system to carry out
- Almost all Linux distro supply a shell program from the  Gnu project called Bash(Bourne Again SHell)

### Terminal Emulators

- When using a GUI(Graphical User Interface) we need another program called *terminal emulator* to interact with the shell for e.g, alacrritty,kitty,kde console etc
- If we have no Terminal Emulators running serveral terminal sessions continue to run behind the graphical desktop we can access these terminal sessions called *virtual terminal*  by pressing `Ctrl+Alt+F1` through `Ctrl+Alt+F6`

### Shell Prompt

- `[me@archlinux ~]$` - basic prompt in bash

- **Some Basic Commands** cal,data,uptime
  - cal - Calander 
  - date - show today's date
  - uptime - uptime of os
  - df - amount of free space in our disk drives 
  - free - amount of free space
