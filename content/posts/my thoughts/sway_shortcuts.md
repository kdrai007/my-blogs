# Sway Configuration Shortcuts
> **Key:** `Super` = Windows Key | `Alt` = Alt Key

## 🚀 Essentials & Launchers
| Key Binding | Action |
| :--- | :--- |
| `Super` + `Enter` | Open Terminal (Ghostty) |
| `Ctrl` + `Alt` + `t` | Open Terminal (Kitty) |
| `Super` + `d` | Application Menu (Fuzzel) |
| `Super` + `Shift` + `d` | Run Command (Wmenu) |
| `Alt` + `g` | Open Google Gemini (Firefox) |
| `Alt` + `c` | Open ChatGPT (Firefox) |
| `Alt` + `F4` | Power Menu |
| `Super` + `v` | Clipboard Manager |
| `Super` + `w` | Wallpaper Selector |
| `Alt` + `w` | Random Wallpaper |

## 🪟 Window Management
| Key Binding | Action |
| :--- | :--- |
| `Super` + `q` | Kill (Close) Window |
| `Super` + `f` | Toggle Fullscreen |
| `Super` + `Shift` + `Space` | Toggle Floating Mode |
| `Super` + `Space` | Switch Focus (Tiling/Floating) |
| `Alt` + `Arrows` | Focus Window (also `h`/`j`/`k`/`l`) |
| `Alt` + `Shift` + `Arrows` | Move Window |
| `Alt` + `b` | Split Horizontal |
| `Alt` + `v` | Split Vertical |
| `Super` + `s` | Stacked Layout |
| `Super` + `e` | Toggle Split Layout |
| `Super` + `Shift` + `-` | Move to Scratchpad |
| `Super` + `-` | Show Scratchpad |

## 🖥️ Workspaces & Monitors
| Key Binding | Action |
| :--- | :--- |
| `Super` + `0-9` | Switch Workspace |
| `Super` + `Shift` + `0-9` | Move Window to Workspace |
| `Super` + `Shift` + `Left`/`Right` | Move Workspace to Monitor |
| `Super` + `Ctrl` + `1` | Move Workspace to Laptop Screen |
| `Super` + `Ctrl` + `2` | Move Workspace to External Monitor |
| `Super` + `p` | Disable External Monitor |
| `Super` + `Shift` + `p` | Enable External Monitor |

## ⚙️ System & Utilities
| Key Binding | Action |
| :--- | :--- |
| `Super` + `Shift` + `c` | Reload Sway Config |
| `Super` + `Shift` + `e` | Logout (Exit Sway) |
| `Super` + `l` | Lock Screen |
| `Super` + `Shift` + `l` | Lock & Suspend |
| `Super` + `Shift` + `w` | Toggle Waybar |
| `Super` + `Shift` + `n` | Toggle Notifications |
| `Ctrl` + `Alt` + `n` | Note Taker Script |
| `Ctrl` + `Alt` + `m` | Music Menu Script |

## 📸 Screenshots
| Key Binding | Action |
| :--- | :--- |
| `Print` | Region Screenshot |
| `Shift` + `Print` | Window Screenshot |
| `Ctrl` + `Print` | Output Screenshot |
| `Ctrl` + `Shift` + `Print` | Full Screen Screenshot |

## 🔊 Media Keys
| Key Binding | Action |
| :--- | :--- |
| `Audio Keys` | Vol Up / Down / Mute / Mic Mute |
| `Brightness Keys` | Brightness Up / Down |
| `Play`/`Pause`/`Next`/`Prev` | Media Controls |
| `Search Key` | Open Wofi |

## 📏 Resize Mode
| Key Binding | Action |
| :--- | :--- |
| `Super` + `r` | **Enter Resize Mode** |
| `Alt` + `Arrows` | Resize Window |
| `Enter` or `Esc` | **Exit Resize Mode** |
